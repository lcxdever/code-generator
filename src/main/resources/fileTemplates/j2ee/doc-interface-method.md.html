Predefined variables will take the following values:
${CLASS_NAME_CANONICAL} the canonical class name of the class, which the method is located
${METHOD_TEXT_FULL} the full text of the method, contains comment, method info etc.
${METHOD_COMMENT_SIMPLE} only the description part of comment
${PARAMETER_TYPES} a list of parameter, Type is "FieldMd", include fields: fieldName, fieldType, canNull, describe
${PARAMETER_TYPE_DESCRIBE} a list of parameter descriptions, Type is "ClassMd", include fields: classNameSimple, fields. the "fields" field is a list, just like the variable "${PARAMETER_TYPES}"
${PARAMETER_EG} the example of parameters
${RETURN_TYPE_NAME_GENERICS_SIMPLE} the simple name of return type with generics
${RETURN_TYPE_DESCRIBE} a list of return type descriptions, just like the variable "${PARAMETER_TYPE_DESCRIBE}"
${RETURN_EG} the example of return
